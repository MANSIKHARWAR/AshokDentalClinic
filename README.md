# Dental Clinic Website

A modern and responsive dental clinic website developed for Ashok Dental Clinic – Orthodontic & Implant Center located in East Vinod Nagar, Delhi.

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Formspree
- Google Maps Embed API

## ✨ Features

- Responsive UI Design
- Appointment Booking Form
- Doctor Profiles
- Treatment Showcase
- Patient Reviews
- Interactive Gallery
- WhatsApp Integration
- Google Maps Location
- Mobile Friendly Navigation

## 📁 Project Structure

├── index.html  
├── style.css  
├── banner.png  
└── README.md

## 🌐 Setup

1. Download or clone the repository
2. Open `index.html` in browser
3. Customize content/images if needed
